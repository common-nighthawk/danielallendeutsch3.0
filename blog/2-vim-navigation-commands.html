Vim Navigation Commands:

sequences you have no excuse not to know

September 1, 2015

I am giving a short talk at the September New York Vim Meetup. It is this Tuesday, September 1. If you're into that sort of thing, you should RSVP.

I will use this page to post the slides and type up the notes after the event. Therefore, pardon the placeholder post in the interim.

The thinking behind the talk is two-fold: 1. I think something that beginners struggle with most is feeling comfortable moving around within a file and 2. I think some more-advanced users stop learning new commands once they get proficient with Vim. But less keystrokes is always better--especially with the basic commands.
